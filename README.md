# Expense Tracker

A modern and responsive **Expense Tracker** built with **React** and **Vite**, using **plain CSS**.  
Track, edit, and delete expenses, view the total amount spent, and persist your data in LocalStorage.

## Features

- Add, edit, and delete expenses  
- Automatic **LocalStorage persistence**  
- Calculate and display **total expenses**  
- Modern UI with cards, shadows, and rounded elements  
- Responsive design for mobile and desktop  
- Editable modal for smoother UX  
- Optional category for each expense  

## Tech Stack

- **React** (functional components + hooks)  
- **Vite** for fast development and HMR  
- **Plain CSS** for modern styling  
- **LocalStorage** for persistence  
- **JavaScript (ES6+)**
